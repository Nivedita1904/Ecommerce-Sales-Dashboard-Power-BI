# Ecommerce-Sales-Dashboard-Power-BI

## Project Overview
This Power BI dashboard analyzes sales, profit, quantity, product performance, and regional trends.

## Tools Used
- Power BI
- Excel
- DAX
- Power Query

## Key Insights
- Identified top-performing products
- Analyzed monthly sales trends
- Compared regional performance
